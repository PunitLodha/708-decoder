# CEA 708 decoder

Currently, parses only binary files(.bin) of [RCWT format](https://github.com/CCExtractor/ccextractor/blob/master/docs/BINARY_FILE_FORMAT.TXT) and ouputs all the parsed commands 

## Usage
```
cargo run <path to bin file>
```
